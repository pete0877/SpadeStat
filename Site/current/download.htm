Download

SpadeStat is currently in BETA testing phase. You may download and install this version, however it will expire on July 1st, 2005.


If you have any questions, feel free to contact us at support@spadestat.com.

  1. Download & Extract: click here to download the main installation file (SpadeStatBeta.zip, 3.7 Mb). Unzip the archive. You will get 5 files out of it.

  2. Install: Simply run the Setup.Exe and follow instructions. If you receive following error during the installation: "This setup requires the .NET Framework...", then follow these sub-steps first to install the .Net framework and retry this step.

    • click here to download Microsoft .Net Framework 1.1 (dotnetfx.exe, 23.1 Mb)

    • Install the .Net Framework by running the downloaded dotnetfx.exe. You will be asked to reboot your computer once installation is done.
Once you are done with SpadeStat installation, you will see new "SpadeStat" shortcut in your "Programs" menu. Run it to begin using SpadeStat.
